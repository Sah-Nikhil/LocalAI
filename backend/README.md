## File Parsing Libraries
| Package       | Purpose                                                                 |
| ------------- | ----------------------------------------------------------------------- |
| `pypdf`       | Extracts text from PDF files                                            |
| `python-docx` | Extracts text from `.docx` (Microsoft Word) documents                   |
| `python-pptx` | Extracts text from `.pptx` (PowerPoint) files                           |
| `lxml`        | Parser backend (used by `python-pptx`) for fast XML parsing             |
| `xlsxwriter`  | You’re not using this directly — likely pulled by `python-pptx` as well |
| `pillow`      | Image support (used indirectly by other parsing libraries)              |
